2024 · BEC Lab (IESL)

Custom Bi-Stable Shutter Driver

Hardware Design Mixed-Signal Testing Laser Optics

The Challenge

Designed and characterized a custom 10-channel output driver module to precisely control TAKANO bi-stable mechanical optical shutters for laser experiments.

Architecture & Testing

The system utilizes a 20-input array (10 control pairs) driving internal H-bridge modules via 13% duty cycle pulses (>5Vp-p). To characterize performance, I built a custom optical test rig passing a master laser through collimators into the shutter, recording the profile on an oscilloscope.

Results

  • Verified perfectly stable electrical frequency response from 1Hz up to 15kHz across all terminals.
  • Characterized mechanical shutter response time at ~25ms.
  • Established absolute maximum operational frequency limits (5 Hz) for experimental safety.
2024 · M.Sc. Thesis (UNIWA)

Autonomous EM Spectrum Analyzer

SDR (RTL-SDR) Raspberry Pi Python & DSP 3D CAD

The Challenge

Developed a low-cost, portable, and autonomous embedded system capable of scanning and recording electromagnetic (EM) radiation across a wide spectrum, while simultaneously logging GNSS location data for spatial RF mapping.

Architecture & Tech Stack

The core processing unit is a Raspberry Pi 4 paired with an RTL-SDR (RTL2832U) module for RF data acquisition, and a SIM7000E NB-IoT Hat for GPS tracking. I wrote a custom Python library ("ListeningEye") to perform Fast Fourier Transforms (FFT) on raw I/Q data, detect active RF bands, and display real-time status on an integrated LCD. The entire assembly is housed in a custom 3D-printed enclosure designed in SOLIDWORKS.

Results

  • Verified system accuracy by running automated Wide Field Scans (WFS) against known commercial FM broadcasting frequencies.
  • Achieved fully autonomous, battery-powered operation with organized CSV data logging for post-processing.
  • Created a cost-effective, portable alternative to traditional spectrum analyzers for field research.
